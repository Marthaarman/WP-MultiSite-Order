== Description ==
Order multisite subsites (wp function get_sites()) with a custom order within the network admin.

== Installation ==

1. Upload [`Multisite Order`](https://github.com/Marthaarman/WP-MultiSite-Order) to the `/wp-content/plugins/` directory

2. Activate the plugin through the 'Plugins' menu in WordPress
